Attributes in dataset are

1. No. of time pregnant
2. Plasma gulcose concentration a 2hrs in an oral gulcose tolerance test
3. Diastoloc blood pressure(mm HG)
4. Triceps skin fold thickness(mm)
5. 2-hrs serum insulin (mm U/ml)
6. Body Mass Index (BMI) (weight in kg/ (height in m)^2)
7.  Diabetes pedigree function
8. Age(years)
9. Output candidate have diabetes or  not represent in (0 or 1)

Pre-requsite libraies
1. tensorflow
2. keras
3. numpy
4. scipy
5. h5py

train.py file is used to train the dataset
test.py file is used to test the data in dataset

Json files were created by running the train file
